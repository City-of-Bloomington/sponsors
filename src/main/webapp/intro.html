Welcome to Sponsorship Database


Please Select one of the options in the left panel.
  • To enter a new sponsor record, select 'New Sponsor'
  • To add a new sponsorship, select New Sponsorship
  • For an extensive search for certain sponsors, select the 'Sponsors' under Search options
  • To enter a new donation record, select 'New Donation'
  • To view certain donations select 'Donations' under Search options
  • When you are done, please Logout from the system to close the session.
